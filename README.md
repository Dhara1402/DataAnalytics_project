# 📊 Data Analytics Projects (SQL-Based EDA & Advanced Analytics)

This repository contains two comprehensive SQL-based data analytics projects focused on exploring datasets, identifying patterns, and generating actionable business insights. The projects demonstrate end-to-end analytical workflows, from data exploration to advanced reporting and performance analysis.

## 🔍 1. Exploratory Data Analysis (EDA)

The goal of this project is to understand the structure, quality, and behavior of the dataset by performing systematic exploratory analysis using SQL.

# 📌 Database Exploration
Examined the overall database structure, including tables, relationships, and data types
Understood how different entities interact within the dataset
Identified primary and foreign key relationships for analysis readiness
# 📌 Dimensions Exploration
Analyzed categorical variables (dimensions) such as customers, products, regions, etc.
Studied unique values and distribution across each dimension
Identified key business segments and categories for deeper analysis
# 📌 Date Exploration
Investigated date fields to understand time-based patterns
Checked data consistency across different time periods
Identified trends, seasonality, and changes over time
# 📌 Measure Exploration
Analyzed numerical metrics using SQL aggregations (SUM, AVG, COUNT, MIN, MAX)
Evaluated key performance indicators (KPIs)
Understood overall business performance through quantitative measures
# 📌 Magnitude Analysis
Assessed the scale and distribution of important business metrics
Identified high-impact and low-impact areas in the dataset
Compared performance across different dimensions
# 📌 Ranking Analysis
Applied ranking techniques using SQL window functions
Identified top-performing and bottom-performing categories
Highlighted key contributors to overall performance
🎯 Outcome:

This analysis provided a strong understanding of the dataset structure and uncovered meaningful insights related to business performance, trends, and category-wise contributions.

## 📈 2. Advanced Data Analytics Project

**  This project focuses on advanced analytical techniques to derive deeper insights and build structured reporting views for business decision-making. **

# 📌 Change Over Time Analysis
Tracked performance metrics across multiple time periods
Identified growth trends, declines, and seasonal variations
Evaluated how business performance evolved over time
# 📌 Cumulative Analysis
Calculated running totals and cumulative metrics using SQL window functions
Analyzed progressive performance trends
Assessed long-term growth patterns
# 📌 Performance Analysis
Compared KPIs across different segments, categories, and time periods
Evaluated business efficiency and performance consistency
Identified underperforming and high-performing areas
# 📌 Part-to-Whole Analysis
Measured contribution of individual segments to overall performance
Determined percentage contribution of categories to total metrics
Helped understand which segments drive the majority of results
# 📌 Data Segmentation
Grouped data into meaningful segments for targeted insights
Analyzed customer and product behavior patterns
Enabled focused decision-making based on segmentation
# 📌 Customer Report
Built customer-level analytical reports
Identified high-value customers and their contribution
Analyzed customer behavior and engagement patterns
# 📌 Product Report
Developed product-level performance reports
Evaluated top-selling and low-performing products
Supported product strategy and optimization decisions
# 🎯 Outcome:

This project delivered deeper business intelligence through structured reporting and advanced SQL analytics, enabling better understanding of customers, products, and overall performance trends.
